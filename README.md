# OmniEFI
Automated OpenCore EFI Builder for AMD and Intel systems with hardware auto-scan
# 🍎 OmniEFI Builder v4.2
**Automated OpenCore EFI Generator for AMD & Intel Systems**

## 🌐 Visão Geral / Overview
**Português:** O OmniEFI é uma ferramenta em Python feita para automatizar a criação de pastas EFI do OpenCore. Ele escaneia seu hardware (CPU, GPU, Rede) e baixa as kexts e drivers mais recentes direto das fontes oficiais.

**English:** OmniEFI is a Python tool that automates OpenCore EFI creation. It scans hardware and downloads the latest kexts and drivers directly from official sources.

## ✨ Funcionalidades / Features
* **Auto-Scan:** Detecta CPUs Ryzen/Intel e placas de rede automaticamente.
* **Smart Download:** Baixa OpenCore, Lilu, VirtualSMC e mais em tempo real.
* **Auto-Update:** O app avisa quando eu lanço uma nova versão aqui no GitHub.
* **Safe Mode:** Configurado para evitar problemas de "Vault" e "SecureBoot" no primeiro boot.

## 🚀 Como Usar / How to Use
1. Baixe o `OmniEFI.exe` na seção **Releases**.
2. Execute como Administrador.
3. Selecione seu Pendrive ou uma pasta.
4. Clique em **"GERAR EFI COMPLETA"**.

## 🛠 Requisitos / Requirements
* Windows 10 / 11 or **Windows PE** (WinPE).
* Conexão com a Internet / Active Internet connection.
* **Privilégios de Administrador / Admin Privileges**.
